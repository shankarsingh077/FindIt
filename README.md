## Recommended Structure For Future Questions

Use one folder per challenge:

```text
FindIt/
├── README.md
├── question-01/
│   └── README.md
├── question-02/
│   └── README.md
├── question-03/
│   └── README.md
```

## Naming Convention

To keep everything easy to locate, future walkthroughs should follow this pattern:

- `question-01/README.md`
- `question-02/README.md`
- `question-03/README.md`

This makes it easy to find:

- the question number
- the walkthrough for that question
- room for extra files later, like screenshots, scripts, or notes
